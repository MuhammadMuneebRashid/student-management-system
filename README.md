# Student Management System

A menu-driven Python application for managing student records using text file storage.

## Features
- Add student records
- View all student records
- Search students by name
- File handling for data storage
- Exception handling

## Technologies Used
- Python
- File Handling
- Functions
- Loops
- Conditional Statements
- Exception Handling

## How to Run
1. Clone the repository.
2. Open the project in VS Code.
3. Run student-management-system.py.
4. Follow the on-screen menu.
